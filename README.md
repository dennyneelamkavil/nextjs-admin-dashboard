# Surprez Admin

**Surprez Admin** is an **ecommerce website admin dashboard** developed by **Ametzo Technologies**.  
It is built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**, providing a modern, scalable, and performant foundation for managing ecommerce operations.

The dashboard is designed to support product management, orders, users, analytics, and administrative workflows in a clean and extensible UI.

---

## Overview

Surprez Admin is a feature-rich admin panel designed for ecommerce platforms, offering reusable components, responsive layouts, and a modern developer experience.

### Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4

---

## Features

- Responsive admin layout with sidebar navigation
- Ecommerce-ready dashboard pages
- Data tables and charts
- Forms, modals, alerts, dropdowns, and buttons
- Authentication-ready UI
- Dark mode support
- Modular and reusable components

---

## Installation

### Prerequisites

- Node.js 18.x or later (Node.js 20.x recommended)

---

### Setup

Clone the repository:

```bash
git clone <your-repository-url>
cd surprez-admin
```

Install dependencies:

```bash
npm install
# or
yarn install
```

Start the development server:

```bash
npm run dev
# or
yarn dev
```

The application will be available at:

```bash
http://localhost:3000
```
